## Build Setup

```bash
# install dependencies
$ npm install

# install Nuxt
$ npm install nuxt

# install axios
$ npm install axios

# plugins
$ npm install sweetalert2
$ npm install --save apexcharts

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
